НАЗНАЧЕНИЕ ЭКСПЕРТА

 

Система «ЭКСПЕРТ» служит для выбора оптимального варианта решения проблемы из нескольких возможных на основе указанных Вами критериев и их важности для Вас. Данная система способна сделать за Вас правильный выбор на основе математических расчетов и Ваших знаний. Решение конкретной проблемы с помощью «Эксперта» будем называть исследованием.

 

 

ШАГИ РЕШЕНИЯ ПРОБЛЕМЫ С ПОМОЩЬЮ «ЭКСПЕРТА»

 

 Для проведения исследования с помощью Эксперта необходимо проделать следующие шаги:

1.      Определить перечень наиболее важных критериев оценки для решения имеющейся задачи

2.      Задать каждому критерию уровень важности. Самый важный критерий будет иметь минимальное число, например «1», следующий за ним по важности – «2», и т.д. Если несколько критериев имеют одинаковую важность, то им присваивается одинаковое число, например два раза «3»

3.      Для исследования берутся несколько вариантов, один из которых будет признан лучшим по сравнению с другими  вариантами

4.      Каждый вариант оценивается оценками по каждому критерию. Каждая оценка должна входить в свой допустимый  предел

5.      Для каждого критерия задается его предел (пороговое значение). Если критерий положительный, то предел задается больше нуля, если отрицательный – то меньше нуля.

 Положительный критерий тот, который изменяется от нуля до своего предела, и чем ближе оценка к пределу, тем лучше. Часто предел положительного критерия задается равным максимальному значению из рассматриваемых вариантов. Отрицательный критерий тот, который изменяется от своего предела до плюс бесконечности, и чем меньше и ближе оценка к своему пределу, тем лучше. Часто предел отрицательного критерия задается равным минимальному значению из рассматриваемых вариантов.

Пример отрицательного критерия: «цена». Этот критерий может иметь значение ”-200”, это будет означать, что цена за товар будет от 200 долларов (рублей, евро и т.д.) и больше. Цена меньше 200 или равная 200 будет считаться идеальной в данном исследовании.

Пример положительно критерия: «скорость ЦПУ». Этот критерий может иметь значение ”2500”, это будет означать, что скорость процессора будет от нуля Мегагерц (мы сами выбираем единицу измерения)  и больше. Скорость больше 2500 или равная 2500 будет считаться идеальной в данном исследовании.

6.      Нажимаете кнопку «Поиск решения» и видите, какой вариант лучший, какой хуже и на сколько процентов

 

 

 

ИТЕРФЕЙС СИСТЕМЫ «ЭКСПЕРТ»

 

 

Управляющий элемент

Назначение

Поле «Тема исследования»

В данном поле нужно ввести тему Вашего исследования, например «Какая программа лучше для напоминания дел ?».

Тема исследования используется как название файла при сохранении исследования на диск (знак «?» при этом автоматически отбрасывается)

Кнопка «Новый проект»

Нажатие на данную кнопку очистит все поля «Эксперта» для работы с новым исследованием. Перед нажатием этой кнопки нужно (при необходимости) сохранить данные

Кнопка «Записать данные в файл»

Данная кнопка служит для записи всех данных текущего исследования в файл с произвольным именем и расширением «dex»

Кнопка «Загрузить данные из файла»

Данная кнопка служит для загрузки в «Эксперт» всех данных ранее сохраненного на диске исследования. Кнопки записи и считывания данных позволяют многократно работать с одним и тем же исследованием, а также копировать его другим пользователям системы «Expert»

Кнопка «Поиск решения»

Самая главная кнопка в системе «Эксперт». Нажатие на эту кнопку заставляет «Эксперта» найти для Вас лучший вариант из имеющихся в данном исследовании

При окончании расчетов «Эксперт» выдаст информационное окно, где будет информация о лучшем варианте, график оценки ВСЕХ вариантов, поле встроенного калькулятора и кнопка «Сохранить график».

Кнопка «Сохранить график» запишет его в указанное вами место, с произвольным именем и расширением «bmp»

Кнопка «Добавить критерий»

Добавляет в конец новый (пустой) критерий. После нажатия на эту кнопку необходимо ввести все необходимые данные для нового критерия

Кнопка «Удалить критерий»

Удаляет последний критерий. После нажатия на эту кнопку критериев становится на один меньше.

P.S. Последовательность критериев в «Эксперте» не имеет значения, т.к. из значимость задается вручную в поле «Важность критерия»

Кнопка «Добавить вариант»

Добавляет вправо новый (пустой) вариант. После нажатия на эту кнопку необходимо ввести все необходимые данные для нового варианта

Кнопка «Удалить вариант»

Удаляет крайний правый вариант. После нажатия на эту кнопку вариантов становится на один меньше.

P.S. Последовательность вариантов не имеет значения, т.к. «Эксперт» оценит варианты и найден лучший вне зависимости от их положения

Поле «Название варианта»

Введите в данном поле нужное название варианта, вместо значения по умолчанию. Например, вместо «Вариант 1» можно написать «Вариант от ТСН», или «Orix» и т.д.

Поле встроенного калькулятора

Служит для проведения произвольных математических расчетов. Используется при размышлении над полученными от «Эксперта» результатами

Кнопка «Выход»

Служит для выхода из системы «Эксперт»

 

 

 

ПРИМЕРЫ РЕШЕНИЯ ПРОБЛЕМЫ С ПОМОЩЬЮ «ЭКСПЕРТА»

 

                  Задача 1:

      Имеются два претендента на определенную работу. Они должны иметь высшее образование и приличные оценки по

точным наукам. Итак, для нас в этом исследовании важны 4 критерия: оценка по математике, по информатике,
     физике, химии  и черчению.

Зададим пределы критериев равным 5 баллам (т.е. от 0 до 5) (оценка в Вузе)

Выставим уровень важности каждому критерию:

·        «оценка по математике» - 2 (менее важный, чем первый)

·        «оценка по информатике»  -1 (самый важный критерий для нас)

·        «оценка по физике» -3

·        «оценка по химии» -3 (одинаково важен, как и «оценка по физике»)

·        «оценка по черчению» -5 (критерий на два уровня меньше, чем предыдущий третий критерий)

 

Оценим Петрова Н.К. (данные возьмем из его диплома):

·        «оценка по математике» - 5 баллов

·        «оценка по информатике»  - 5 балла

·        «оценка по физике» - 3 балла

·        «оценка по химии» - 4 баллов

·         «оценка по черчению» - 3 балла

 

Оценим Сидорова К.М. (данные возьмем из его диплома):

·        «оценка по математике» - 4 баллов

·        «оценка по информатике»  - 5 балла

·        «оценка по физике» - 4 балла

·        «оценка по химии» - 5 баллов

·         «оценка по черчению» - 3 балла

 

Кто из них нам больше подходит, учитывая заданные критерии и их важность для нас ?

 

Введя данные в систему «Эксперт» получаем: Петров: 86%, Сидоров: 88%. Отсюда видно, что лучшим в данном  исследовании оказался Сидоров на 2 %. Данный демо - пример можно загрузить и посмотреть через кнопку «Загрузить данные из файла».

 

Следует отметить, что решение Эксперта основывается на Ваших оценках, критериях и пределах значений. При изменении какого-либо параметра (из вышеуказанных) может оказаться, что другой вариант лучше, чем найденный вариант.

 

Если Эксперт Вам сказал, что лучший вариант имеет 100%, то это значит следующее: данный вариант полностью удовлетворяет Всем указанным Вами требованиям на основе проставленных Вами оценок в рамках указанных критериев. В конечном счете, Вы ответственны за решения Эксперта.

 

Правильное решение Эксперт может дать только человеку, являющемуся специалистом в исследуемой области, т.к. только он может указать нужные критерии, расставить их по важности, определить их предельное значение и дать адекватную оценку по каждому варианту.

 

                  Задача 2:

Какая программа лучше для напоминания дел ?

Имеем следующие критерии и задаем им важность и предел:

·        «Количество функций (5 базовых функций)» - важность «2», предел «5»

·        «Простота работы (0-5 баллов)» - важность «1», предел «5»

·        «Время установки на компьютере, мин» - важность «3», предел «-10» (от 10 минут и больше)

·        «Цена, руб» - важность «4», предел «-200» (от 200 рублей и больше)

 

Оценим программу «ORIX Profile»

·        «Количество функций (5 базовых функций)» - 3

·        «Простота работы (0-5 баллов)» - 5

·        «Время установки на компьютере, мин» - 1

·        «Цена, руб» - 130

 

Оценим программу «Microsoft Outlook»

·        «Количество функций (5 базовых функций)» - 4

·        «Простота работы (0-5 баллов)» - 3

·        «Время установки на компьютере, мин» - 10

·        «Цена, руб» - 6000

 

Введя данные в систему «Эксперт» получаем: «Orix»: 88%, «Microsoft Outlook»: 68%. Отсюда видно, что лучшим в данном исследовании оказался «Orix» на 20 %. Данный демо - пример можно загрузить и посмотреть через кнопку «Загрузить данные из файла».
